1. Clone the repository:
   ````
   git clone https://github.com/OrF8/PEPSE.git
   cd PEPSE
   ````
2. Import the project into your IDE:
   - For IntelliJ IDEA:â€‹
     - Open IntelliJ IDEA.

  - For Eclipse:â€‹
     - Open Eclipse.
     - Select "File" > "Import" > "Existing Projects into Workspace".
